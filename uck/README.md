# DESCRIPTION

Installs [UCK - Ubuntu Customization Kit](http://uck.sourceforge.net/). UCK is a tool that helps you customizing official Ubuntu Live CDs (including Kubuntu/Xubuntu and Edubuntu) to your needs. You can add any package to the live system, for example language packs, or applications.


# REQUIREMENTS

## Supported Platforms

The following platforms are supported by this cookbook, meaning that the recipes run on these platforms without error:

* Ubuntu

# RECIPES

* `default`, Default recipe

# USAGE

This cookbook installs the UCK components if not present, and pulls updates if they are installed on the system.
